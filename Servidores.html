Software de Servidores de Internet

Um servidor W3, como o daemon ftp, é um programa que responde a uma conexão tcp de entrada e fornece um serviço ao chamador. Existem muitas variedades de software de servidor W3 para servir diferentes formas de dados.

O programa de Teste

O programa de teste W3 básico serve arquivos já em hipertexto ou texto simples. Esse programas é usado como base para muitos outros tipos de servidor e gateway. A documentação inclui:

  • Guia do usuário
    • .
  • Internals -- uma descrição do código.

    • Muito do acima também pode ser aplicado no todo ou em parte a outros servidores mencionados abaixo. Seja qual for o servidor que você está executando, você provavelmente estará interessado em:

  • Ferramentas para provedores de informação

    • Fazendo um novo servidor

    Este programa é frequentemente usado como base para um servidor mais específico para uma determinada aplicação. Um servidor que permite que um mundo de dados seja visto como parte do universo W3 é conhecido como gateway. (A maioria dos servidores pode, portanto, ser considerada como gateways, mas o termo implica alguma conversão ou mapeamento entre mundos diferentes). Para tutoriais curtos com exemplos, consulte:

  • Escrevendo um servidor como um script

    • É uma boa ideia escolher o daemon básico ou um dos servidores abaixo como ponto de partida ao criar um novo servidor.

    Outros servidores e gateways

    São servidores que fornecem dados extraídos de outros sistemas. eles são construídos usando código do daemon básico, ou scripts.

    ENCONTRAR gateway
    para CERN/VM XFIND que chama um REXX exec para obter as informações do sistema XFIND em execução no mainframe CERNVM.
    Gateway de ajuda do VMS
    Isso permite que qualquer arquivo de ajuda do VMS seja disponibilizado para clientes WWW. Funciona em VAX/VMS.
    WAISGate
    Um gateway para informações disponíveis usando o protocolo WAIS.

    Servidor DCL

    Um servidor para sistemas VMS que permite escrever um gateway para seu próprio sistema de informação favorito usando DCL.

    System33

    Um (grande) servidor de script csh que fornece dados, incluindo documentos Xerox System33, páginas man em texto simples, números de telefone, etc. etc...!

    Oráculo

    Um servidor genérico para oracle. Pode ser usado como base para gateways para bancos de dados Oracle específicos.

    Geografia

    Porta de entrada para o Servidor de geografia na U Michigan
